# CineBuddy — Onde trabalhar

**Sempre abra esta pasta no Cursor:**

```
E:\_Projetos\Cinebuddy\CineBuddy-V13
```

**Como abrir:**  
No Cursor: **File** → **Open Folder** → navegue até o disco **E:** → `_Projetos` → `Cinebuddy` → `CineBuddy-V13` → **Selecionar Pasta**.

Assim o Cursor usa as regras do projeto (`.cursor/rules`) e você edita e envia para o GitHub a partir de um único lugar.

Para conectar ao GitHub e fazer o primeiro envio, use o arquivo **`CONECTAR_ESTE_PROJETO_AO_GITHUB.md`**.
